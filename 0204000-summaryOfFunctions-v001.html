16  16 Summary of Functions

Numeric Functions

AVERAGE(number1,[number2],…)

Returns the arithmetic mean of the supplied numbers. Text and empty cells are ignored.

  • number1 – The first number, cell reference, or range to average.
  • [number2], … – Optional additional numbers, ranges, or references to include in the average.

Example: =AVERAGE(A1:A10) returns the mean of values in A1 through A10.

SUM(number1,[number2],…)

Adds numbers together and returns their total.

  • number1 – The first number, cell reference, or range to add.
  • [number2], … – Optional additional numbers, ranges, or references to include in the total.

Example: =SUM(A1:A10) adds all values in A1 through A10.

Counting Functions

COUNT(value1,[value2],…)

Counts the number of cells containing numeric values.

  • value1 – The first value, cell reference, or range to evaluate.
  • [value2], … – Optional additional values or ranges to include.

Example: =COUNT(A1:A10) counts how many cells in the range contain numbers.

COUNTA(value1,[value2],…)

Counts the number of non-empty cells (numbers, text, logical values, or errors).

  • value1 – The first value, reference, or range to evaluate.
  • [value2], … – Optional additional values or ranges to include.

Example: =COUNTA(A1:A10) counts all cells in the range that are not empty.

… also see COUNTIF and COUNTIFS in the “…IF/…IFS” functions section below

max, min, large, small functions

MAX(number1,[number2],…)

Returns the largest numeric value in a list or range.

  • number1 – The first number, reference, or range.
  • [number2], … – Optional additional numbers or ranges.

Example: =MAX(A1:A10) returns the largest value in the range.

MIN(number1,[number2],…)

Returns the smallest numeric value in a list or range.

  • number1 – The first number, reference, or range.
  • [number2], … – Optional additional numbers or ranges.

Example: =MIN(A1:A10) returns the smallest value in the range.

LARGE(array,k)

Returns the k-th largest value in a dataset.

  • array – The range or array containing numeric values.
  • k – The position from largest (1 = largest value, 2 = second largest, etc.).

Example: =LARGE(A1:A10,2) returns the second-largest value.

SMALL(array,k)

Returns the k-th smallest value in a dataset.

  • array – The range or array containing numeric values.
  • k – The position from smallest (1 = smallest value, 2 = second smallest, etc.).

Example: =SMALL(A1:A10,3) returns the third-smallest value.

Logical Functions

IF(logical_test,[value_if_true],[value_if_false])

Performs a logical test and returns one value if the test is TRUE and another if FALSE.

  • logical_test – A condition that evaluates to TRUE or FALSE (e.g., A1>10).
  • [value_if_true] – Value returned if the test is TRUE.
  • [value_if_false] – Value returned if the test is FALSE.

Example: =IF(A1>50,"Pass","Fail")

IFS(logical_test1 , value_if_true1 , … )

Evaluates multiple conditions and returns the value associated with the first TRUE condition.

  • logical_test1 – First condition to evaluate.
  • value_if_true1 – Value returned if the first condition is TRUE.
  • Additional logical tests and results can follow.

Example: =IFS(A1>=90,"A",A1>=80,"B",A1>=70,"C")

AND(logical1,[logical2],…)

Returns TRUE if all supplied conditions evaluate to TRUE.

  • logical1 – First logical condition.
  • [logical2], … – Optional additional logical conditions.

Example: =AND(A1>0,A1<100)

OR(logical1,[logical2],…)

Returns TRUE if any of the supplied conditions evaluate to TRUE.

  • logical1 – First logical condition.
  • [logical2], … – Optional additional logical conditions.

Example: =OR(A1="Yes",A1="Y")

IFERROR(value,value_if_error)

Returns a specified value if a formula results in an error.

  • value – The formula or expression to evaluate.
  • value_if_error – Value returned if an error occurs.

Example: =IFERROR(A1/B1,"Division Error")

“IS” Functions

ISBLANK(value)

Checks whether a cell is empty.

  • value – The cell or expression to test.

Returns TRUE if the cell is blank, otherwise FALSE.

ISERROR(value) … also se IFERROR in the Logical Functions section

Checks whether a value results in an error.

  • value – The value or formula to test.

Returns TRUE if the result is any Excel error (e.g., #DIV/0!, #N/A).

ISEVEN(number)

Checks whether a number is even.

  • number – The number to evaluate.

Returns TRUE if the number is even, otherwise FALSE.

ISODD(number)

Checks whether a number is odd.

  • number – The number to evaluate.

Returns TRUE if the number is odd, otherwise FALSE.

“…IF/…IFS” functions

SUMIF(range,criteria,[sum_range])

Adds values that meet a specified condition.

  • range – Cells evaluated against the condition.
  • criteria – The condition used to determine which cells to include.
  • [sum_range] – Optional range of values to sum; if omitted, range is summed.

Example: =SUMIF(A1:A10,">10",B1:B10)

SUMIFS(sum_range,criteria_range1,criteria1,…)

Adds values that meet multiple conditions.

  • sum_range – The range of numbers to sum.
  • criteria_range1 – First range to evaluate.
  • criteria1 – Condition applied to the first range.
  • Additional criteria ranges and conditions may follow.

AVERAGEIF(range,criteria,[average_range])

Calculates the average of values that meet a specified condition.

  • range – Range evaluated against the condition.
  • criteria – Condition used to select cells.
  • [average_range] – Optional range containing the values to average.

AVERAGEIFS(average_range,criteria_range1,criteria1,…)

Calculates the average of values that meet multiple conditions.

  • average_range – Range containing numbers to average.
  • criteria_range1 – First range evaluated.
  • criteria1 – Condition applied to the range.

COUNTIF(range,criteria)

Counts the number of cells that meet a specified condition.

  • range – Cells to evaluate.
  • criteria – Condition used to count matching cells.

Example: =COUNTIF(A1:A10,">=50")

COUNTIFS(criteria_range1,criteria1,…)

Counts cells that meet multiple conditions.

  • criteria_range1 – First range evaluated.
  • criteria1 – Condition applied to the range.
  • Additional range–criteria pairs can follow.
Date and Time Functions

TODAY()

Returns the current system date.

This function takes no arguments and updates automatically when the worksheet recalculates.

DATE(year, month, day)

Creates a valid Excel date from separate year, month, and day values.

  • year – Four-digit year.
  • month – Month number (1–12).
  • day – Day of the month.

Example: =DATE(2026,3,4)

MONTH(serial_number)

Returns the month (1–12) from a date.

  • serial_number – A valid Excel date or reference containing a date.

YEAR(serial_number)

Returns the year from a date.

  • serial_number – A valid Excel date or reference.

WEEKDAY(serial_number)

Returns the day of the week as a number.

  • serial_number – A valid Excel date.

By default: 1 = Sunday, 7 = Saturday.

EDATE(start_date, months)

Returns a date a specified number of months before or after a given date.

  • start_date – Starting date.
  • months – Number of months to add (negative values subtract months).

Example: =EDATE(A1,6)

TEXT(value, format_text)

Converts a number or date into text using a specified format.

  • value – The number or date to format.
  • format_text – Formatting code (e.g., "mm/dd/yyyy").

Example: =TEXT(A1,"mmmm yyyy")

Lookup Functions

VLOOKUP(lookup_value,table_array,col_index_num,FALSE)

Searches the first column of a table and returns a value from a specified column.

  • lookup_value – Value to search for.
  • table_array – Range containing the lookup table.
  • col_index_num – Column number (within the table) to return a value from.
  • FALSE – Specifies an exact match.

Example: =VLOOKUP(A2,A1:D100,3,FALSE)

XLOOKUP

A modern replacement for VLOOKUP that searches a range and returns a matching value from another range.

Basic syntax: XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], …)

  • lookup_value – Value to search for.
  • lookup_array – Range containing values to search.
  • return_array – Range containing values to return.
  • [if_not_found] – Optional value returned if no match is found.

Example: =XLOOKUP(A2,A2:A100,B2:B100)

Textual Functions

CONCATENATE(text1,[text2],…) (similar to & operator)

Joins multiple text strings into a single string.

  • text1 – First text value.
  • [text2], … – Optional additional text values.

Example: =CONCATENATE(A1," ",B1)

LEN(text)

Returns the number of characters in a text string.

  • text – The text or cell reference to evaluate.

LEFT(text,[num_chars])

Returns characters from the beginning of a text string.

  • text – Text string to extract from.
  • [num_chars] – Number of characters to return (default = 1).

RIGHT(text,[num_chars])

Returns characters from the end of a text string.

  • text – Text string to extract from.
  • [num_chars] – Number of characters to return.

MID(text,start_num,num_chars)

Extracts characters from the middle of a text string.

  • text – Original text string.
  • start_num – Position of the first character to extract.
  • num_chars – Number of characters to return.

FIND(find_text, within_text, [start_num])

Finds the position of one text string within another.

  • find_text – Text to locate.
  • within_text – Text to search within.
  • [start_num] – Optional starting position for the search.
Random Numbers functions

RAND( )

Returns a random decimal number between 0 and 1.

This function takes no arguments and recalculates each time the worksheet updates.

RANDBETWEEN(bottom, top)

Returns a random integer within a specified range.

  • bottom – The smallest possible number.
  • top – The largest possible number.

Example: =RANDBETWEEN(1,100)
