21  Spill Ranges

This tutorial introduces spill ranges in Excel: dynamic formula outputs that automatically fill neighboring cells. You will learn how to create spill ranges, reference them with the # operator, summarize them, format them, and retrieve specific values, rows, and columns with INDEX.

Note

This tutorial is based on the following video:
Working with Spill Ranges in Excel: Basic to Advanced Techniques by BrainBell.
https://www.youtube.com/watch?v=4ehggj5D6Bg

21.1 Learning objectives

By the end, you will be able to:

  • Explain what a spill range is.
  • Fix common #SPILL! problems.
  • Create spill ranges with SEQUENCE and SORT.
  • Reference an entire spill range using the # operator.
  • Use standard functions such as SUM and COUNT on spill ranges.
  • Use INDEX to retrieve individual values, columns, rows, and limited result sets.

21.2 Prerequisites

You need a version of Excel that supports dynamic arrays, such as Microsoft 365 or recent versions of Excel.

21.3 Sample setup

Create a small table like this in Excel:

Name Department Sales
Ava East 120
Ben West 95
Cara East 140
Diego South 110
Eli West 160
Fatima North 105
Grace South 130

Assume the table is named SalesData.

To turn a range into a table:

  1. Select the data.
  2. Press Ctrl + T.
  3. Check My table has headers.
  4. Rename the table to SalesData from the Table Design tab.

21.4 What is a spill range?

A spill range is the group of cells filled by a formula that returns more than one result.

For example, this formula returns a vertical list of numbers:

=SEQUENCE(10)

Enter the formula in one cell. Excel automatically fills the cells below it with the results.

Tip

Only the top-left cell contains the actual formula. The other cells display results produced by that formula.

21.5 Fixing the #SPILL! error

A #SPILL! error usually means Excel cannot place the formula results into the needed output cells.

Common causes include:

  • Data already exists in the spill destination.
  • A merged cell blocks the output.
  • The spill range would extend beyond the worksheet.
  • The formula is inside an Excel table cell where dynamic spilling is restricted.

To fix it:

  1. Select the cell showing #SPILL!.
  2. Look for the dashed outline showing where Excel wants to spill the results.
  3. Clear or move anything blocking that area.
  4. Unmerge any merged cells in the spill destination.
  5. Re-enter or refresh the formula.

21.6 Create a spill range with SEQUENCE

The SEQUENCE function creates a list or grid of numbers.

Basic vertical sequence

=SEQUENCE(10)

This returns numbers 1 through 10 in a single column.

Sequence with rows and columns

=SEQUENCE(5, 3)

This returns 5 rows and 3 columns.

Sequence with start and step values

=SEQUENCE(5, 1, 10, 5)

This returns:

10
15
20
25
30

The arguments are:

=SEQUENCE(rows, [columns], [start], [step])

21.7 Create a spill range with SORT

The SORT function can return a dynamic sorted version of a range or table.

Sort the sales table by the Sales column:

=SORT(SalesData, 3, -1)

Explanation:

  • SalesData is the source table.
  • 3 means sort by the third column.
  • -1 means descending order.

This creates a spill range containing the sorted table.

21.8 Reference a spill range with the # operator

When a dynamic array formula begins in a cell, you can reference its entire spill range by adding # to the formula cell reference.

For example, if this formula is in F2:

=SORT(SalesData, 3, -1)

Then this reference points to the full spilled result:

=F2#

This is useful because the reference expands or shrinks automatically when the spilled result changes.

21.9 Summarize a spill range

Suppose F2# contains a spilled table with sales in the third column.

Count rows in the spill range

=ROWS(F2#)

Count cells in the spill range

=COUNT(F2#)

Sum the third column of the spill range

=SUM(INDEX(F2#, , 3))

The empty row argument in INDEX means “return all rows.”

21.10 Format a spill range

You can format a spill range similarly to a normal range:

  1. Click inside the spilled result.
  2. Select the full spill range.
  3. Apply number formatting, borders, font styling, or fill colors.
Important

Formatting the cells is separate from the formula. If the spill range grows, new cells may not always inherit formatting exactly as expected. Using source tables and conditional formatting is more reliable.

21.11 Apply conditional formatting to a spill range

To highlight sales greater than 125:

  1. Select the spill range, such as F2#.
  2. Go to Home > Conditional Formatting > New Rule.
  3. Choose Use a formula to determine which cells to format.
  4. Use a formula like:
=$H2>125
  1. Choose a format and apply the rule.

In this example, column H is assumed to contain the spilled Sales values.

21.12 Retrieve values with INDEX

The INDEX function is one of the most useful tools for working with spill ranges.

The general pattern is:

=INDEX(spill_range, row_number, column_number)

If your spill range starts in F2, use:

=INDEX(F2#, row_number, column_number)

21.13 Retrieve the first value in the first column

=INDEX(F2#, 1, 1)

This returns the value in the first row and first column of the spill range.

21.14 Retrieve the first value in the second column

=INDEX(F2#, 1, 2)

This returns the value in the first row and second column.

21.15 Extract the first two columns

To return all rows from the first two columns:

=INDEX(F2#, , SEQUENCE(,2))

Explanation:

  • The blank row argument returns all rows.
  • SEQUENCE(,2) returns column numbers {1,2}.
  • INDEX returns columns 1 and 2 from the spill range.

21.16 Pull specific rows and columns

To return rows 1 through 3 and columns 1 and 3:

=INDEX(F2#, SEQUENCE(3), {1,3})

This gives a smaller dynamic extract from the spilled result.

Tip

Use array constants such as {1,3} when you want specific non-adjacent columns.

21.17 Dynamically retrieve rows with SEQUENCE and INDEX

To return the first 5 rows of a spill range:

=INDEX(F2#, SEQUENCE(5), )

To return the first 5 rows and first 2 columns:

=INDEX(F2#, SEQUENCE(5), SEQUENCE(,2))

This pattern is helpful when you want to limit a larger spilled result.

21.18 Sort and return only the top 5 results

To sort SalesData by sales and return the top 5 rows:

=INDEX(SORT(SalesData, 3, -1), SEQUENCE(5), )

This combines:

  • SORT to order the data.
  • INDEX to select rows.
  • SEQUENCE(5) to request the first five rows.

21.19 Top 5 names and sales only

To return only the Name and Sales columns for the top 5 rows:

=INDEX(SORT(SalesData, 3, -1), SEQUENCE(5), {1,3})

This produces a compact leaderboard.

21.20 Practice exercises

Exercise 1

Create a sequence from 1 to 20.

=SEQUENCE(20)

Exercise 2

Create a 4-row by 3-column sequence.

=SEQUENCE(4,3)

Exercise 3

Sort SalesData by Sales from largest to smallest.

=SORT(SalesData, 3, -1)

Exercise 4

Assume the sorted result starts in F2. Count the rows in the spill range.

=ROWS(F2#)

Exercise 5

Return the first three rows of the sorted spill range.

=INDEX(F2#, SEQUENCE(3), )

Exercise 6

Return only the first and third columns from the first three rows.

=INDEX(F2#, SEQUENCE(3), {1,3})

21.21 Common mistakes

Mistake Why it happens Fix
#SPILL! appears The output area is blocked Clear the cells in the spill destination
Formula only returns one value The formula is not returning an array Use a dynamic array function such as SEQUENCE, SORT, FILTER, or UNIQUE
A reference does not expand A normal range reference was used Use the spill operator, such as F2#
INDEX returns the wrong field The row or column number is wrong Count rows and columns carefully, including headers
Formatting does not expand cleanly New spilled cells may not inherit formatting Use conditional formatting or format a larger expected area

21.22 Key takeaways

Spill ranges make Excel formulas more dynamic. Instead of copying formulas across many cells, you can enter one formula and let Excel return all needed results.

The most important patterns are:

=F2#

Reference the full spill range.

=INDEX(F2#, 1, 1)

Retrieve one value.

=INDEX(F2#, , 3)

Retrieve one full column.

=INDEX(F2#, SEQUENCE(5), )

Retrieve the first five rows.

=INDEX(SORT(SalesData, 3, -1), SEQUENCE(5), {1,3})

Create a sorted top-five report.

21.23 Suggested next steps

Try replacing SORT with other dynamic array functions:

=UNIQUE(SalesData[Department])
=FILTER(SalesData, SalesData[Sales]>120)
=SORT(FILTER(SalesData, SalesData[Sales]>120), 3, -1)

These formulas build on the same spill-range concepts.
