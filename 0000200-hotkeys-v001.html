17  “hot-keys”

Excel keyboard “hot-keys” can make you MUCH more productive.

  • On Macs, the Command (⌘) key is used for most keyboard shortcuts that would typically use the Ctrl key on Windows.

  • On Macs, the Option (⌥) key is used for most keyboard shortcuts that would typically use the Alt key on Windows.

  • Mac keyboards have a “Return” key. Windows keyboards have an “Enter” key. These are basically equivalent.

17.1 The hotkeys

The list of shortcuts below was adapted from https://corporatefinanceinstitute.com/resources/excel/excel-shortcuts-pc-mac/

The list below is arranged in order of “importance”. The importance info is based on my personal thoughts. I categorized the importance levels as follows:

importance meaning
0 Basic and “obvious” (these are common in many other programs too)
1 Every Excel user should know these
2 Extremely useful to know
3 Helpful, but not so much
4+ Might not be worth memorizing but could be helpful. Can usually do these by pressing buttons.
